# Sistema de colores

###### Colores primarios

Esta escala de colores representa al color principal del sistema, con el cual se deben resaltar las acciones principales dentro de la interfaz de una manera sutil y equilibrada. Para no generar elementos de distracción dentro de una pantalla. La aplicación de esta escala debe estar relacionada con elementos tales como: Botones primarios, radio buttons, checkboxes, textos destacados, componentes de carga, etc.

###### Colores semánticos

Los colores semánticos son muy importantes dentro del sistema porque ayudaran al usuario a comprender la respuesta del sistema ante una acción determinada. Las tonalidades **azules** sirven para indicar alguna información importante sobre un proceso o acción, el color **rojo** indicara los errores que puede generar el proceso o acción y que pueden ser destructivos, el **amarillo** advierte sobre un proceso o acción que no es destructiva y por ultimo el color **verde** que ayuda al usuario a entender cuando un proceso finaliza con éxito.
